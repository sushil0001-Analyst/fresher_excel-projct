# 📊 Loan Data Analysis Dashboard (Excel)

## 🧠 Project Overview
This project analyzes loan application data to understand repayment behavior, risk factors, and borrower profiles using Microsoft Excel.

---

## 🎯 Objective
To identify patterns affecting loan repayment and provide insights for better lending decisions.

---

## 📂 Dataset Information

The dataset includes:

- Borrower profile details
- Employment information
- Credit grade
- Loan status
- Homeownership type
- Last payment date
- Next payment due

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques
- Dashboard Visualization

---

## 🧹 Data Cleaning Steps

✔ Removed duplicates  
✔ Standardized employment length format  
✔ Handled missing values  
✔ Organized columns for analysis  

---

## 📊 Key Insights

- Loan status distribution (Fully Paid vs. Unpaid)
- Impact of employment length on repayment
- Credit grade risk comparison
- Homeownership trends among borrowers

---

## 📈 Dashboard Features

- Interactive charts
- Summary KPIs
- Category-wise analysis
- Easy-to-understand visuals
